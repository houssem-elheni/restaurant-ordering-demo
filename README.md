# Restaurant Ordering Demo

A full customer + restaurant ordering system using Firebase.

## Features

- Customer app:
  - Browse pizzas, sandwiches, drinks
  - Add extras (fromage, harissa, olives, sauce blanche, scaloppe)
  - Dynamic price calculation
  - Place an order and see status updates

- Restaurant app:
  - Receive new orders in real time
  - Approve orders (customer sees "Approved")
  - Mark orders as "Ready" (customer sees "Ready")
